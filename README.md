# Grafana Plugins

This repository serves as a signing repository for Grafana plugins, such as [mongodb-datasource](https://github.com/haohanyang/mongodb-datasource).

## Purpose

This repository is used to manage and sign Grafana plugins. It is intentionally kept minimal and serves primarily as a container for plugin signing operations.

## Related Plugins

- [mongodb-datasource](https://github.com/haohanyang/mongodb-datasource) - MongoDB data source plugin for Grafana
